Kairos：把握时间的质量，而非数量————在规律与灵感之间，找到生活的节奏；
古希腊的两种时间，在古希腊，人们用两个截然不同的词来描述时间：
Chronos（χρόνος） 是钟表上的时间——滴答滴答，日复一日，不紧不慢地流逝。它是量化的、线性的、无情的。我们说"时间都去哪儿了"，说的就是 Chronos。它像沙漏里的细沙，无论你如何挽留，都会一粒粒滑落。
Kairos（καιρός） 则完全不同。它是"恰当的时机"，是"关键时刻"，是那些让生命有了意义的瞬间。当你灵光一闪想到绝妙创意的那一刻，当你终于下定决心改变的那个清晨，当你与挚友深夜长谈感到被理解的那个瞬间——这些，都是 Kairos。
古希腊人深知：人生的意义不在于经历了多少个 Chronos 的小时，而在于捕捉到了多少个 Kairos 的瞬间。
